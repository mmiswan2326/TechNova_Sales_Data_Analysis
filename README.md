# 📊 TechNova Sales Data Analysis

## Project Overview

This project was completed as part of the TechNova Data Analytics Internship Program.

The objective of this project is to analyze retail sales data from the Superstore dataset and generate business insights using data analytics and visualization techniques.

---

## 📁 Dataset Information

Dataset: Sample Superstore Dataset

Dataset Statistics:

- Total Records: 9,994
- Total Features: 21

The dataset contains information about:

- Orders
- Products
- Customers
- Regions
- Categories
- Sales
- Profit

---

## 📊 Key Results

### Revenue and Profit

- Total Revenue: $2,297,200.86
- Total Profit: $286,397.02

### Top Category by Sales

- Technology: $836,154.03

### Best Performing Region

- West: $725,457.82

### Best Performing Product

- Canon imageCLASS 2200 Advanced Copier
- Sales: $61,599.82

---

## 💡 Business Insights

- Technology generated the highest sales revenue among all categories.
- The West region recorded the highest overall sales performance.
- Revenue is concentrated among a small number of high-performing products.
- Top-performing products should receive additional marketing and inventory support.
- Regional sales performance varies significantly across different markets.
