# TechNova Sales Data Analysis

## Project Objective

Analyze sales performance using the Superstore dataset and generate business insights through data analytics and visualization.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## KPIs Analyzed

- Total Revenue
- Total Profit
- Sales by Category
- Sales by Region
- Top Products

## Key Results

- Total Revenue: $2,297,200.86
- Total Profit: $286,397.02
- Top Category: Technology
- Top Region: West
- Top Product: Canon imageCLASS 2200 Advanced Copier
